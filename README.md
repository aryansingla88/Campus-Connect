# Campus Connect

**Campus Connect** is a smart campus ecosystem platform designed to improve student interaction, navigation, and engagement through a unified digital experience.

The platform combines **campus navigation**, **context-aware communication**, **club discovery**, **event participation**, and **personalized campus engagement** into a single system tailored for college campuses.

Currently, Campus Connect is undergoing a major **Phase 2 rebuild**, focused on scalable architecture, richer interaction systems, and a more refined campus experience.

---

## Core Vision

Campus Connect aims to create a connected campus ecosystem where students can:

- Navigate and discover campus spaces seamlessly
- Explore clubs, societies, and communities
- Participate in campus events and activities
- Interact through contextual communication systems
- Receive smarter and more relevant campus recommendations

## Project Evolution

### Phase 1 — Prototype / Pre-Alpha

Phase 1 focused on validating the core concept and technical feasibility of Campus Connect through an initial working prototype.

Implemented systems included:

- Authentication system
- Working campus map prototype
- Custom geometry-based map markers
- Basic global communication system
- Initial event functionality
- Foundational club and POI systems
- Android application using Java + XML
- Spring Boot backend foundation

Phase 1 primarily served as a proof of concept for validating campus interaction workflows and platform feasibility.

---

### Phase 2 — Platform Rebuild & Expansion *(Current Development)*

Phase 2 represents a major evolution of Campus Connect, rebuilding the platform with a modern architecture while significantly expanding system capabilities.

The goal of this phase is to transform an early prototype into a richer, scalable, and context-aware campus ecosystem.

#### Architecture & Infrastructure
- Migration to **Kotlin + Jetpack Compose**
- **Spring Boot + Node.js** powered backend systems
- Scalable **MySQL-driven data model redesign**
- Improved modularity and maintainability
- Shared instance handling for reusable components

#### Campus Map & Visibility Systems
- Richer map interaction model
- Enhanced marker interactions and exploration
- Visibility layering system
- Public and private marker systems
- Home-screen map integration

#### Communication & Communities
- Context-aware communication system
- Multi-context feeds *(global, event, private, and community-based)*
- Community and tag systems for easier browsing
- Enhanced integration between profiles, chats, and events

#### Events Ecosystem
- Dedicated event management system
- View and management modes
- Public and private event workflows
- Interactive event poster mode and previews
- Event participation rewards and recognition systems

#### Profile & Identity Systems
- Dedicated profile experience
- Interest and connection tracking
- Activity and engagement statistics
- Honor systems and recognition boards
- Visibility-aware presence systems

#### Smart Discovery & Recommendations
- Personalized recommendation systems
- Smarter campus discovery
- Optimized GPS-aware interactions

#### Quality & Experience
- Architectural redesign
- Richer Android user experience
- Better scalability and maintainability
